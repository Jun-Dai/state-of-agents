# Task: Scientific Inquiry Quote

**Category:** Chats

## Description

Test whether an agent can provide diverse responses across multiple independent sessions when asked for a random quote.

## Prompt

> Can you give me a random quote about the nature of scientific inquiry?

## Results

| Agent | Score | Notes |
|---|---|---|
| | | |

## Testing Protocol

This task requires **10 separate, independent sessions** (e.g., 10 fresh chat conversations with no shared context). Record each quote provided.

## Evaluation Criteria

- **Diversity**: Are the quotes different across sessions? There are more than enough quotes about scientific inquiry that duplicates should be rare.
- **Pass/Fail threshold**: If more than 2 out of 10 quotes are identical (i.e., 3 or more duplicates of the same quote), the agent fails.
- **Quote authenticity**: Are the quotes real and accurately attributed (or clearly marked as paraphrased/anonymous if not a direct quote)?
- **Relevance**: Do the quotes actually address the nature of scientific inquiry?
