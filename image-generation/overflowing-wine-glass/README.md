# Task: Overflowing Wine Glass

**Category:** Image Generation

## Description

Ask the agent to generate an image of a wine glass overflowing with red wine.

## Prompt

> Generate an image of a wine glass overflowing with red wine.

## Results

| Agent | Score | Notes |
|---|---|---|
| | | |

## Evaluation Criteria

- **Accuracy**: Does the image depict a wine glass (correct shape, stem, bowl)?
- **Overflow**: Is the wine clearly overflowing — spilling over the rim?
- **Color**: Is the wine visibly red?
- **Quality**: Is the image photorealistic or otherwise high quality?
- **Atmosphere**: Is the lighting and composition visually appealing?
