# Task: Transposition

**Category:** Sheet Music

## Description

Ask the agent to transpose a piano piece up a whole step (major second interval).

## Evaluation Criteria

- **Correctness**: Are all notes transposed up by a whole step (e.g., B♭ → C, E♭ → F)?
- **Key signature**: Is the key signature correctly adjusted (B♭ major with 3 flats → C major with no accidentals)?
- **Rhythm preservation**: Are all rhythmic values preserved exactly?
- **Articulation and dynamics**: Are markings like staccato, dynamics (mf, dim.), and expression markings (Andante cantabile) preserved?
- **Format**: Is the output in a usable format (rendered notation, LilyPond, MusicXML, ABC notation, or similar)?
- **Completeness**: Is the entire piece transposed, including all measures and the ornamentation examples at the bottom?
